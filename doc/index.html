Stip

Stip 适用于页面提示效果。兼容主流浏览器（ie,firefox,chrome,opera,Safari 等），不依赖其他框架，不使用图片（三角是用CSS做的）

Hello world

参数是字符串，默认Stip没有关闭按钮。

Stip('test').show("hello World");

关闭按钮

参数为一个json数据格式，这里配置内容和关闭按钮

Stip('test1').show({
	content:"hello world",
	closeBtn:true
});

任意位置

Stip 可以在4个方向弹出提示框，参数又p来配置

Stip('test2').show({
	content:"hello world",
	closeBtn:true,
	p:'left'
});


Stip('test3').show({
	content:"hello world",
	closeBtn:true,
	p:'top'
});


Stip('test4').show({
	content:"hello world",
	closeBtn:true,
	p:'bottom'
});


Stip('test5').show({
	content:"hello world",
	closeBtn:true,
	p:'right'
});

多样式

默认为你提供了两种样式，可根据情况自己 定义更多样式

Stip('test6').show({
	content:"hello world",
	closeBtn:true,
	p:'top',
	kind:'error'
});

Stip('test7').show({
	content:"hello world",
	closeBtn:true,
	p:'top',
	kind:'ok'
});
