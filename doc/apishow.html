Stip

Stip 适用于页面提示效果。兼容主流浏览器（ie,firefox,chrome,opera,Safari 等），不依赖其他框架，不使用图片（三角是用CSS做的）

Hello world

参数是字符串，默认Stip没有关闭按钮。

Stip('test').show("hello World");

关闭按钮

参数为一个json数据格式，这里配置内容和关闭按钮

Stip('test1').show({
	content:"hello world",
	closeBtn:true
});

任意位置

Stip 可以在4个方向弹出提示框，用参数p来配置

Stip('test2').show({
	content:"hello world",
	closeBtn:true,
	p:'left'
});


Stip('test3').show({
	content:"hello world",
	closeBtn:true,
	p:'top'
});


Stip('test4').show({
	content:"hello world",
	closeBtn:true,
	p:'bottom'
});


Stip('test5').show({
	content:"hello world",
	closeBtn:true,
	p:'right'
});

多样式

默认为你提供了两种样式，可根据情况自己 定义更多样式

Stip('test6').show({
	content:"hello world",
	closeBtn:true,
	p:'top',
	kind:'error'
});

Stip('test7').show({
	content:"hello world",
	closeBtn:true,
	p:'top',
	kind:'ok'
});

设定显示时间

显示一秒，一秒以后自动清除

Stip('test8').show({
	content:"hello world",
	time:1000
});

把提示信息写在元素上

.show方法默认是一个function,他会读取当前元素的 data-stip 属性 作为提示内容

<button id="test11" data-stip="从test11读取的提示内容">Click here</button>
							
Stip('test11').show();//.show没有参数

参数是一个function

.show 可以传递一个function,注意function的第一个参数将会接收到当前element，function返回值作为提示内容

Stip('test12').show(function(ele){
	return ele.getAttribute('stip');
});

显示和隐藏

用代码清除提示，先保存stip对象，调用它的show,和hide方法

(function(){
	var stip = null;
	$('#test9').click(function(){
		stip = Stip('test9');
		stip.show({
			content:"显示提示效果",
			p:'left'
		});
	});
	
	
	$('#test10').click(function(){
		stip.hide();
	});
})();
//上面使用了jquery  
//也可以像下面这样
var stip = Stip('test9');
stip.show("显示提示效果"); //显示
stip.hide();//隐藏
