Parameter String(element id)
Parameter Element

.show

显示提示

Parameter none //相当于传递了Funciont参数
	-function(ele){return ele.getAttribute("data-stip");}							
Parameter String //提示内容
Parameter Function //返回值作为提示内容，默认接受 element参数
Parameter JSON
	-json = {
		content: String, // 默认 空
		p:String //('left' | 'top' | 'right' | 'bottom'), 默认right
		kind:String //('ok' | 'error'), 默认 ok
		time: Number// 显示时间(单位毫秒), 默认不关闭
		closeBtn:Boolean// 关闭按钮 默认 false
		
	}
return  false;

配置更多提示样式

如果你不太满意ok error这两种样式，你完全可以按照CSS中 ok的样式去修改。


/* ok css */
.lj-ok{color:#000; border:1px solid #2192D3; background:#F4FBFF;}
.lj-ok .lj-top{border-top-color:#2192D3;}
.lj-ok .lj-right{border-right-color:#2192D3;}
.lj-ok .lj-left{border-left-color:#2192D3;}
.lj-ok .lj-bottom{border-bottom-color:#2192D3;}

.lj-ok .lj-top .lj-span{border-top-color:#F4FBFF;}
.lj-ok .lj-right .lj-span{border-right-color:#F4FBFF;}
.lj-ok .lj-left .lj-span{border-left-color:#F4FBFF;}
.lj-ok .lj-bottom .lj-span{border-bottom-color:#F4FBFF;}

/* 依照 ok css 写一个xxoo的CSS*/
/* xxoo CSS */
.lj-xxoo{color:#000; border:1px solid #f30; background:#000;}
.lj-xxoo .lj-top{border-top-color:#f30;}
.lj-xxoo .lj-right{border-right-color:#f30;}
.lj-xxoo .lj-left{border-left-color:#f30;}
.lj-xxoo .lj-bottom{border-bottom-color:#f30;}

.lj-xxoo .lj-top .lj-span{border-top-color:#000;}
.lj-xxoo .lj-right .lj-span{border-right-color:#000;}
.lj-xxoo .lj-left .lj-span{border-left-color:#000;}
.lj-xxoo .lj-bottom .lj-span{border-bottom-color:#000;}

//注意  border:1px solid #2192D3; border的颜色和lj-top,lj-right,lj-left,lj-bottom颜色一致
//background:#F4FBFF; 颜色和 .lj-span的颜色保持一致
