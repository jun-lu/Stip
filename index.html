Stip说明。
        Stip是一个简单的提示组建，我期望它可以把表单验证变得简单。
        简单，小巧，灵活。
    

    3.0 更新说明
    1: 添加静态Stip.config.conteng配置，这里你可以全局配置它成为一个function,返回值将作为提示的内容.
    2: show 方法的参数 可以直接传递一个 function|String|Number|json map.
    3: show 方法 json map 的.content 也可以是一个function.
    看下面的例子
    

        
        基本使用方法,点击测试
        // 基本使用方法
        Stip('test5').show({
        	content:"用户名已经存在",
            kind:"error",
            time:5000
        });
        

        Stip('id') 会返回一个对象 里面包含了 .show(),.hide() 方法
        所以你可以这样使用它
        
        var tip = Stip('id');
        tip.show({content:"提示内容"}); // 显示提示
        tip.hide()// 关闭提示
    
    	
        
        Stip.config 的可配置清单。
        1：下面的的参数都可以作为 show方法的 参数传递
        2：也都可以用过 Stip.config.XXX = XXX,来实现全局配置
        
        Stip.config = {
                p: 'right', // String [left|top|right|bottom] 默认弹出框的方向 
                kind: 'correct', // String [correct|error] 提示的类型(提示的颜色不同) 正确和错误
                closeP: 'ljClose', // String  关闭按钮前缀, 一般你不需要改变
                closeBtn: false, // boolean 默认是否有关闭按钮
                time:null, // Number 提示框显示时间, null 一直显示
                content:function(){ return "Hello World";},// function|String 默认提示内容，如果是function this会指向当前 DOM对象， 第一个参数也是当前DOM对象 
            }
        

    
    	配置所有 弹出提示有关闭按钮， 上面的配置清单都可以用下面方式配置
        
    	Stip.config.closeBtn = true;
        
    

    	3.0 版本新添加的 content 为function的使用方法。
        注: function 的this 指针指向当前DOM对象，第一个参数也是当前dom 对象,
            方法返回值作为提示的内容
        
    	直接传递function,点击测试
        
        
    	Stip('test2').show(function(){ return this.getAttribute('Stip');});  // 直接在show 方法传递一个方法 function
        
        
        
        
    	全局配置了 content,p(提示内容，提示位置)，点击测试
        	
        	
        Stip.config.p = "top";  // 弹出位置在上方
        
        // 提示内容是对象的 Stip属性	
        Stip.config.conteng = function(){ 
        	return this.getAttribute('Stip');
        }
        
    	Stip('test3').show();  // 直接调用了 show 方法而不用传递任何参数
    

    	进一步了解它
    	拷贝下面某一段代码去执行
        
    	Stip('testCode').show(function(){ return this.id;});
        
        Stip('testCode').show("直接传递字符串");
        
        Stip('testCode').show({content:'1秒消失的提示', time:1000});
        
        Stip('testCode').show({content:'提示在上方', p:"top"});
        
        Stip('testCode').show({content:'错误提示，一般颜色偏红', kind:"error"});

        tip = Stip('testCode'); tip.show("推荐保存Stip对象,需要用到的时候再调用show方法");
        tip.hide();// 需要它消失的时候调用hide() 方法
